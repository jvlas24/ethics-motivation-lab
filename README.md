## 🚀 Deploy to GitHub Pages (3 steps)

1. **Create a new repository** on GitHub (e.g. `ethics-motivation-lab`)
2. **Upload all files** — drag & drop this entire folder into the repo
3. Go to **Settings → Pages → Deploy from branch → main → / (root)** → Save

Your site will be live at `https://[your-username].github.io/ethics-motivation-lab/`

## 📋 Embedding activities in Blackboard Ultra

Each activity is a **single self-contained HTML file** — no external dependencies.

1. Open the activity page on the website
2. Click **"Descargar HTML para Blackboard"**
3. In Blackboard Ultra: **Create Document → + → HTML** → paste → Save
4. Students interact directly inside the LMS
