Design principles · Principios de diseño

Six principles that ground the framework

Seis principios que anclan el marco

01
Pedagogical coherence
Coherencia pedagógica
The medium embodies the message. A course about gamification must itself be a live gamified experience.
02
Ethical grounding
Anclaje ético
Every gamification mechanic carries values. Each design choice is made explicit and examined critically.
03
Intrinsic motivation
Motivación intrínseca
Growth logic replaces punitive grading. XP accumulation reflects learning, not compliance.
04
Dual-layer evaluation
Evaluación de doble capa
Competency progression tracks mastery; participation monitoring guides real-time instructor support.
05
AI as inquiry object
IA como objeto de indagación
AI is both a pedagogical tool and a subject of critical ethical analysis throughout the course.
06
Full Blackboard Ultra integration
Integración total en Blackboard Ultra
Native tools as infrastructure. No external platforms, no plugins — everything lives inside the LMS.

Activity library · Biblioteca de actividades

Interactive HTML activities, ready to embed

Actividades interactivas HTML listas para embedder en Blackboard Ultra

Each activity is a single HTML file generated with Anthropic Claude and embedded directly inside Blackboard Ultra. No external tools, no plugins — the entire interaction happens within the LMS. Download any activity, open in a text editor, and paste the code into Blackboard's HTML editor.

How to embed in Blackboard Ultra

  1. Open any activity page below
  2. Click "Download HTML"
  3. In Blackboard Ultra, create a new Document
  4. Click the "+" → HTML → paste the code
  5. Save — students interact without leaving the LMS
